# 1xBet Mobcash Telegram Bot

This is a simple Telegram bot for handling 1xBet Mobcash transactions.

## How to Deploy

1. Install dependencies:  
   ```sh
   pip install -r requirements.txt
   ```
2. Run the bot:  
   ```sh
   python bot.py
   ```

## Deployment on Render

1. Push this repository to GitHub.
2. Connect GitHub to Render.
3. Set the environment variable `YOUR_BOT_TOKEN_HERE` with your bot's token.

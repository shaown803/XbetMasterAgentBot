# 1xBet Mobcash Telegram Bot

This is a simple Telegram bot for handling 1xBet Mobcash deposits and withdrawals.

## Setup Instructions

1. Replace `YOUR_BOT_TOKEN` in `bot.py` with your actual Telegram Bot Token.
2. Install dependencies using `pip install -r requirements.txt`.
3. Run the bot using `python bot.py`.

For deployment on Render or Railway, follow their respective documentation.

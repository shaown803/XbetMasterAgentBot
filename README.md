# Xbet Master Agent Bot

This bot automates the deposit and withdrawal process for 1xBet Mobcash players using Telegram.

## Features:
- **Deposit System**: Players request deposit availability, send money, and get approval.
- **Withdrawal System**: Players request withdrawal, submit details, and get confirmation.
- **Manual Approval**: Admin manually verifies and approves transactions via a private Telegram group.

## How to Use:
1. Players send deposit/withdrawal requests.
2. The bot provides necessary details.
3. Admin approves or rejects requests.
